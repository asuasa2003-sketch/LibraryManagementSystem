📚 Library Management System

Overview

The Library Management System is a command-line application developed using Python and SQLite. It allows users to efficiently manage books, library members, and book loan records through a simple and user-friendly interface.

This project demonstrates the use of Python programming, SQL database management, and Git/GitHub version control in a real-world application.

⸻

Features

📖 Book Management

* Add new books
* View all books
* Search for books
* Update book information
* Delete books

👥 Member Management

* Add new members
* View all registered members

🔄 Loan Management

* Record book loans
* Store loan information in the database

How to Run

1. Clone the repository:

git clone https://github.com/asuasa2003-sketch/LibraryManagementSystem.git

2. Navigate to the project folder:

cd LibraryManagementSystem

3. Run the application:

python3 main.py
