# Library Management System

## Project Description
This project is a Library Management System developed using Python, SQLite, SQL, and GitHub.

The system allows users to:

- Add books
- View books
- Search books
- Update books
- Delete books

## Technologies Used

- Python
- SQLite
- SQL
- GitHub

## Files

- main.py
- database.py
- schema.sql
- library.db

## How to Run

```bash
python3 main.py
